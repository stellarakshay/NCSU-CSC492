  • How to Authorize?

Connecting to GitHub will allow Concert to automatically monitor your students' repositories and activity. The data analyzed will include their commit infromation, comments, and code.

Step 1: Create an github Organization

First, begin by creating a GitHub organization for the course that is included on for all the students' repositories.You have to be the owner of students' repositoies in order to add them into your organization.

After you create the organization,go to the setting page of that organization, click developer settings->oauth apps->new oauth app.

Step 2: Create an Oauth application

Then, fill out the form like the following, and click register application:

Step 3: Add Oauth information to Concert

Copy the client id and client secrete from the Github and go back to Concert, enter those information and the name of your organization, then click add. You will be redirected to the Github authorization page, click authorize to finish the authorization process.
